# Node.js_SNS
Node.js를 이용한 간단한 SNS 개발 
<hr/>

## 사용목록

- JavaScript (ES6+)<br><br>
- AJAX <br><br>
- PUG (Jade)<br><br>
- Node.js (Express)<br><br>
- Sequelize (Node.js ORM (Mysql))<br><br>
- Passport.js <br><br>
